## Getting Started

```shell
git clone git@github.com:NimaAIMLDL/segmentation-app.git 
cd segmentation-app
python3 -m venv .venv
. .venv/bin/activate
python3 -m pip install --upgrade pip
pip install -r requirements.txt
python3 app.py
```
